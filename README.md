# perl-cli-ugly-number

## Description
Divide the number by greatest divisible powers of
2, 3 and 5, if the number becomes 1 then it is an
ugly number otherwise not.

## Tech stack
- perl 5

## Docker stack
- alpine:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
